# Simple Interest Calculator

## Project Overview
This project contains a Bash script designed to calculate simple interest. It is a part of the IBM "Introduction to Git and GitHub" course project.

## Script Details
The script `simple-interest.sh` computes simple interest based on the following user inputs:
1. **Principal Amount**: The initial amount of money.
2. **Rate of Interest**: The annual interest rate (as a percentage).
3. **Time Period**: The duration in years.

## Formula
The calculation is based on the standard formula:
Simple Interest = (Principal × Rate × Time) / 100
