# github-final-project


## Description
A simple shell script project developed as part of the IBM Git and GitHub course. This project demonstrates basic version control skills, including forking, branching, and merging.

## Project Details
* **Course:** Introduction to Git and GitHub
* **Final Project Part:** Part 1 & Part 2
* **Script Function:** The `simple-interest.sh` script calculates simple interest based on Principal, Rate, and Time.

## Files in this Repository
* `README.md`: Project documentation and details.
* `LICENSE`: Apache 2.0 License.
* `CODE_OF_CONDUCT.md`: Guidelines for community interaction.
* `CONTRIBUTING.md`: Instructions for contributors.
* `simple-interest.sh`: The core Bash script for interest calculation.
