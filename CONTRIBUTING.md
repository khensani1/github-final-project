## How Can I Contribute?

All contributions, bug reports, bug fixes, documentation improvements, enhancements, and ideas are welcome.

### Reporting Bugs
If you find a bug, please open an issue and include details about your environment and steps to reproduce the error.

### Bug Fixes
If you have a fix for an existing bug, please submit a Pull Request. Ensure your code follows the existing style of the project.

### Documentation Improvements
Documentation is just as important as code. If you find typos or want to clarify instructions in the README, feel free to submit a change.

### Enhancements and Ideas
Have an idea to make the simple interest calculator better? We would love to hear it! Open an issue to discuss your suggested enhancement.

## Pull Request Process
1. Fork the repository.
2. Create a new branch for your feature or fix.
3. Submit a pull request with a clear description of your changes.
